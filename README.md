# KazakhCharsReshaper
Kazakh Arabic Chars Reshaper JQuery Plugin
###Reshaper Web Kazakh Chars
    $(function () {
            $.reshaperWebKzChars();
    })
###Reshaper Kazakh Chars
        var str = $.reshaperKzChars("سالەم");
###Reshaper Kazakh Chars
        var str = $.disReshaperKzChars("ﺳﺎﻟﻪﻡ");
